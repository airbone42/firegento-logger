SyslogMessage::GetMessageChunks

Returns the chunks that form the full log message. If the message is short enough, it's not split and just returned.

    Signature

    public function GetMessageChunks()

    Returns

    An
    array containing the chunk of the Log Message. If the message is
    short enough, the array will contain only one value.
