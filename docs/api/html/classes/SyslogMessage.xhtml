    Methods

    private

    • GetMessageID() — Generates a unique ID for the message. This function is used only if the message is too long and has to be split into chunks; the ID generated by this method can be used to trace the pieces and put them back together.
    • GetPriority() — Calculates and returns the Priority of a Log Message.
    • GetSyslogTimestamp() — Returns a Timestamp string, formatted as required by RFC3164.
    • GetTag() — Builds the TAG part of the Syslog Message.

    protected

    • Coalesce() — Returns the first non null value amongst the arguments.
    • FormatMessage() — Puts all Log Message elements together to form a string that will be passed to the RSysLog Server.
    • GetFQDN() — Returns the Fully Qualified Domain Name of the Server.
    • GetHostName() — Returns the Host Name.
    • GetIPAddress() — Returns the IP Address of the Server.
    • GetProcessName() — Returns the Process Name.
    • SetOptions() — Sets several options for the instance.

    public

    • __construct() — Class constructor.
    • GetMessageChunks() — Returns the chunks that form the full log message. If the message is short enough, it's not split and just returned.
    • SetFacility() — Setter for Facility property.

    • SetProcessName() — Setter for ProcessName property.
    • SetSeverity() — Setter for Severity property.
